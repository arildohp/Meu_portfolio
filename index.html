Projetos em Destaque

De ideias abstratas a projetos bem-sucedidos, aqui estão alguns dos meus trabalhos mais recentes

  • Lista de contatos

    O projeto consiste em uma lista de contatos desenvolvida com React e TypeScript, com o objetivo de oferecer aos usuários uma maneira eficiente de gerenciar suas informações de contato. Possui funcionalidades como adicionar, editar, excluir e buscar contatos, além de ter sido implementado com foco na tipagem estática para maior robustez no código. Foram utilizadas tecnologias como React, TypeScript, HTML5, CSS3, hooks (useState, useEffect) e Styled Components.

  • Desenvolvimento de Loja Virtual Eplay

    O projeto é um site de uma loja de games desenvolvida em React e TypeScript, com foco em proporcionar uma experiência de usuário intuitiva e responsiva. As principais funcionalidades incluem um catálogo de jogos, carrinho de compras. Foram utilizadas tecnologias como useState, useEffect, Context API, Styled Components e integração com APIs.
